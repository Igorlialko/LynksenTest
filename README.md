## Getting Started

Project was created with node v16.16.0

Install dependencies:

```bash
npm install
```

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

For build project in dev:

```bash
npm run build:dev
```
